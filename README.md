# 情報セキュリティ脅威の40年史

IPA「情報セキュリティ10大脅威 2026」を軸に、コンピュータウイルスの黎明期（1986年）から
AI時代（2026年）まで、サイバー脅威の歴史を **時系列（ドキュメンタリー風タイムライン）** でたどる Web サイトです。

スクロールすると映像のように年代が流れ、「自動再生」ボタンでスライドショーのように進みます。

## 特徴

- **時系列スクロール演出**：年代ごとに背景色が変化し、上部に「現在の年」が流れる
- **自動再生（▶）**：操作なしで映像のように歴史を再生
- **IPA 10大脅威 2026**：〈組織〉〈個人〉の一覧（初選出年・継続回数つき）
- **数字で見る個人の脅威**：フィッシング・クレカ不正・ネットバンキング不正送金などの実数
- **「昔とどう違うのか」比較**：動機・規模・標的・手口・影響・境界の6軸で昔↔今を対比
- 各事件・統計に **出典リンク** を付与
- 依存ライブラリなし（Vanilla HTML / CSS / JS）。静的サイトとしてそのまま動作。

## ファイル構成

```
index.html              … 本体
assets/css/style.css    … デザイン
assets/js/data.js       … 脅威の年表・IPAデータ・統計（内容はここを編集）
assets/js/main.js       … スクロール演出・自動再生
.nojekyll               … GitHub Pages の Jekyll 処理を無効化
```

## ローカルで開く

`index.html` をブラウザで開くだけ。ビルド不要です。
（フォントは Google Fonts を読み込むため、表示時はネット接続を推奨）
